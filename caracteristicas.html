Caracteristicas del software



·Citas veterinarias

El software gestiona consultas veterinarias mediante la asignación de citas médicas (Vacunación, desparasitación etc.)




·Citas estéticas

El software gestiona servicios de estética para tu mascota mediante citas.




·Gestión de guardería

El software gestiona servicios de guardería para mascotas.




·Recetas médicas

El sistema permite configurar recetas médicas para que se generen automáticamente al aplicar un tratamiento.




·Gestionar documentos

El sistema gestiona documentos (expedientes, recetas médicas, carnet de vacunación, autorizaciones, etc ). Además de almacenar y gestionar los historiales médicos de los pacientes.




·Comunicación con el propetario

Con VetMex puede mantener una relación estrecha con los propietarios de las mascotas. Desde el propio sistema les puede enviar recordatorios de vacunas, campañas promocionales, mensajes directos, etc.




·Gestión de tareas

El apartado de gestión de Tareas permite asignar tareas y funciones a los diferentes usuarios de la clínica optimizando el tiempo del equipo.




·Hospitalización

El módulo de hospitalización le permitirá recibir, coordinar y controlar las mascotas hospitalizadas. VetMex permite gestionar diferentes áreas de jaulas, validaciones diarias de medicaciones, cargos de hospitalización y mucho más.
